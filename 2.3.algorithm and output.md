2.3 Simple Menu Driven Calculator

Scenario: Perform basic operations using user menu selection.

Algorithm:

1. Input two numbers, a and b.

2. Display menu: 1) Add, 2) Subtract, 3) Multiply, 4) Divide.

3. Input user choice.

4. Use if or switch logic for operation:

   • If choice is 1, print a+b
   • If choice is 2, print a-b
   • If choice is 3, print a*b
   • If choice is 4 and b != 0, print a/b, else print error
   • Else print invalid choice

Output:
Enter a number: 8
Enter a number: 2
1. Add
2. Subtract
3. Multiply
4. Divide
Choose (1/2/3/4): 1
Result: 10.0
