2.5 Prime Number Check

Scenario: Input a number and check whether it is prime.

Algorithm:

1. Input a number n.

2. If n <= 1, it is not prime.

3. For i from 2 to n-1, if n % i == 0, it is not prime.

4. If no such i found, it is prime.

Output:

Enter a number: 7

Prime number

Enter a number: 9

Not a prime number
